# Video Game Player Statistics Analysis System

## 📖 Description

A comprehensive Streamlit application that analyzes video game player statistics and visualizes performance metrics. This tool helps gaming communities understand gameplay trends, compare player performance, and identify skill development patterns.

## ✨ Features

- 📊 Player statistics dashboard
- 📈 Performance charts and graphs
- 🔍 Search players by name
- 🏆 Compare multiple players
- 📉 Interactive visualizations
- 📁 CSV file upload support
- 🎮 Gaming stats analysis
- 📊 Trend analysis

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Plotly
- Seaborn

## 🚀 Installation

### Prerequisites
- Python 3.7+
- pip (Python package manager)

### Steps

```bash
# Clone the repository
git clone https://github.com/Manojkelam/Video-Game-Player-Stats-Analysis.git

# Navigate to project directory
cd Video-Game-Player-Stats-Analysis

# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run app.py
```

## 📊 Dashboard Sections

### Player Dashboard
- Overall statistics
- Performance metrics
- Win/Loss ratio
- Average scores
- Play time analysis

### Performance Charts
- KDA (Kill/Death/Assist) trends
- Win rate over time
- Score progression
- Rank changes
- Achievement unlocked

### Player Search
- Search by username
- View player profile
- Career statistics
- Seasonal performance
- Achievement badges

### Player Comparison
- Compare up to 5 players
- Head-to-head statistics
- Performance metrics side-by-side
- Strength and weakness analysis

## 📈 Supported Metrics

- **KDA:** Kills, Deaths, Assists
- **Performance:** Win rate, Average score
- **Experience:** Total play time, Levels
- **Achievements:** Badges earned, Milestones
- **Rankings:** Current rank, Rank history
- **Consistency:** Deviation, Streaks
